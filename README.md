## worker-pool

Примитивный worker-pool с возможностью динамически добавлять и удалять воркеры. Входные данные (строки) поступают в канал, воркеры их обрабатывают (например, выводят на экран номер воркера и сами данные). Задание на базовые знания каналов и горутин.

#### `make build`

Создаёт приложение в bin/redditclone.

#### `make test`

Запускает тесты.

#### `make lint`

Запускает линтер.

## Запуск контейнера

```bash
docker compose -f build/docker-compose.yml up -d --build
```
